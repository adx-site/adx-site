Midjourney User Guide - Quick Start (1)

July 28, 2024 Midjourney 本文总阅读量次

Goal: Learn how to interact with the Midjourney Bot on Discord through text commands to create images from simple text prompts.

Join Discord

Go to http://Midjourney.com, select Join Beta, or directly go to Midjourney Discord.

Before joining the Midjourney Discord server, you must log in to Discord. Learn how to create a Discord account.

Add Newbie Channel

In the Midjourney official server, select any newbies-# channel visible on the left sidebar. You can also create your own server and invite the Midjourney Bot to your server.

Use the /imagine Command

About Discord Commands

Use commands to interact with the Midjourney Bot on Discord. Commands can:

  • Create images
  • Change default settings
  • Retrieve your user information
  • Other functions

The command’s function is to generate a unique image from a short text description (called a Prompt).

Using the /imagine Command

  1. Enter the command from the slash command popup window
  1. Or quickly select the command
  1. Enter the description of the image you want to create in the “prompt” input box.
  2. Click return to send the message.

If you do not see the popup window when entering the command

, log out and log back in.

The command is only valid in bot channels. It will not work in other channels such as #trial-support.

Processing Drawing Tasks

Midjourney Bot takes about a minute to generate four images.

The platform offers 20 minutes of free task time, approximately enough to generate 25 images; after the free quota is used up, a paid subscription is required.

Each image task includes:

  • Using
  • Command to create images (batch of 4)
  • Upscaling images
  • Image-to-image (regenerating based on the current image)

All of these operations consume your account’s quota; one generation task takes about 1 minute.

Use

command to quickly check your account’s remaining task time.

Upscale and Add Details or Create Variations

After the initial image grid is generated, two rows of buttons will appear:

The U buttons upscale the image, generating a larger version of the selected image and adding more details.

The V buttons create slight variations of the selected grid image. Creating variations generates a new grid of images with the overall style and composition of the selected image.

Refresh reruns the task. In this case, it will rerun the prompt and generate a new grid of images.

Rate Images

After using the upscale image, a new set of options will appear:

Make Variations: Create variations of the upscaled image and generate a new grid with four options.

Beta/Light Upscale Redo: Redo the upscale using different upscaler models.

Link: Open the image in the gallery on Midjourney.com

Rate images by clicking a smiley face button on the Midjourney website or on Discord. Every day, the top 1000 image raters receive one hour of free fast mode time. Active raters also receive bonus task time.

Save Images

Click on the image to open it in full size, then right-click and select

. On mobile, long-press the image, then tap the download icon in the upper right corner.

All your created images can be downloaded from midjourney.com/app, so you don’t have to worry about losing them.

Subscription Plans

Trial users can generate about 25 images for free; the quota does not expire but does not renew. To create more images, use the command

to generate a link to the Midjourney account page.

For pricing and more information, visit Subscription Plans.

Additional Notes

Operating in Your Own Channel

Messages in the official channel are numerous and refresh quickly; you can create your own channel. Images generated in your own channel are subject to the same content and review rules and can be viewed on the Midjourney website.
