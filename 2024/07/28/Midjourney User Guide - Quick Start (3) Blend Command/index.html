Midjourney User Guide - Quick Start (3) Blend Command

Jul 28 2024 2 minutes read (About 244 words)

The /blend command allows you to quickly upload 2-5 images, view the concept and aesthetics of each image, and merge them into a novel new image.

/blend Command

After entering the /blend command, the system will prompt you to upload two photos. When using a mobile device, drag and drop images from your hard drive or add images from your photo library. To add more images, select the optional/options field and choose image3, image4, or image5. The /blend command may take longer to start than other commands because the images must be uploaded before Midjourney Bot can process them.

The default aspect ratio for blended images is 1:1, but you can choose between square (1:1), portrait (2:3), or landscape (3:2) aspect ratios using the optional dimensions field.

A custom suffix is added to the end of the /blend prompt, just like any other /imagine prompt. The aspect ratio specified as part of the /blend command will override the aspect ratio in the custom suffix.

Tips for Use

For best results, upload images with the same aspect ratio as the desired outcome.
