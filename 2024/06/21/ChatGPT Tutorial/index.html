ChatGPT Tutorial

Jun 21 2024 10 minutes lesen (Über 1443 Wörter)

ChatGPT has been with us for over a year, yet its popularity remains high. If you haven’t had a chance to interact with AI, you might think it’s very difficult to learn how to use it.

Everyone says that AI is powerful and can greatly change our lives and work, but in reality, getting started is not as hard as you might think. The key is finding the right way to begin.

This article will help you understand the basics of using ChatGPT, including the registration process, prompt writing, and more.

How to Register for ChatGPT for Free

Visit the ChatGPT official website (https://chat.openai.com) and click the “Sign up” button. You can choose to register with an email address; using a Google email is recommended.

You can also choose to log in directly through Google, Microsoft, or Apple accounts.

If you register with an email, the system will prompt you to set a password, which must be at least 12 characters long.

After completing the registration, you will receive an email with a link to verify your email address. Afterward, the system might ask for additional information, such as your full name and date of birth.

Once these steps are completed, you will have successfully registered!

For a more detailed registration tutorial, please see: Detailed ChatGPT Registration Tutorial 2024: A Comprehensive Step-by-Step Guide!

Log in to ChatGPT, and at the bottom of the main interface, you will find an input box where you can start conversing with the AI.

So, what should we do after registering for ChatGPT? Here are some usage tips…

How to Use ChatGPT

When you start using ChatGPT or other AI platforms, you will often come across the term “prompt.” A prompt is essentially a command you give the AI, telling it what you want it to help you with.

For example, “write a script to process data,” and it will immediately generate a script for you.

However, if you use a more specific prompt, such as “write a detailed document explaining a new feature: write a Python script to read data from a CSV file, filter out rows with missing values, and output the results to a new CSV file, ensuring the script can handle more than 10,000 rows of data and run on both Windows and Linux systems,” you will get content that better meets your needs.

You can write your own prompts as needed or use prompts written by others.

When writing your own prompts, it is crucial to be clear about what you want ChatGPT to accomplish. Using directive verbs such as “write,” “explain,” “create,” etc., can make your commands more precise. The clearer your instructions, the more likely you are to get results that meet your expectations.

Additionally, providing enough context information is key. As mentioned earlier, a vague instruction might yield a script, but it might not meet your specific needs.

Make sure you specify what you want ChatGPT to include (or exclude). For example, if you don’t want your script to include date data, make sure to state that clearly.

When interacting with ChatGPT, expressing your true feelings or problems can help you get better answers.

For example, saying “this issue is really important for my work” or “I really need an accurate answer this time” can help ChatGPT better understand your needs and provide more satisfactory answers.

If you get an answer from ChatGPT that you like but it’s not perfect, you don’t need to start over.

Simply tell the AI what went wrong, and it will try again, considering your corrections. Something like “Can you rewrite the above content, but this time only use 12 lines instead of 16?”

Providing ChatGPT with more information about yourself can also help you get more targeted answers.

Note: Do not provide personal privacy information, such as when using it to help write an email. Telling ChatGPT you are a project manager can help it provide more focused and accurate responses.

In fact, you can provide ChatGPT with some detailed information about yourself in advance, so it can make personalized adjustments when answering your questions each time.

To do this, visit the “Customize ChatGPT” option by clicking on the personal information icon in the bottom left corner of the page. Here, you can add information about yourself and customize how you want the platform to respond.

For example, if you are a student, you can fill in “I am a diligent college student majoring in English” in the first box of the customization description and add “Always reply in a formal academic writing style and provide detailed explanations, avoiding unnecessary wording” in the second box.

How to Manage How ChatGPT Uses Your Data

ChatGPT reminds users not to share too much personal information. This is good advice for using any AI tool, but especially important when using ChatGPT.

You should never share personal data, sensitive company information, or password information with ChatGPT.

This year, OpenAI introduced a memory feature for ChatGPT, which is not yet available to all users.

As the name suggests, this feature allows the tool to remember information about you for future use.

If you are uncomfortable with this, you can choose to turn it off. Go to “Settings” > “Personalization” > “Manage Memory,” where you can view and delete specific memories or clear all memory.

Just like your web browser, ChatGPT remembers everything you previously entered. But you can also manage this information. Under “Settings” > “General,” you can view all previous chat records and delete them individually. You can also use a red button to delete all records at once.

Is It Worth Upgrading to ChatGPT Plus

If you are just starting to use ChatGPT, the free version should be sufficient. However, if you are already proficient in using ChatGPT, then a subscription service might be more suitable for you.

The Plus subscription costs $20 per month. With the upgrade, you can use the more powerful ChatGPT model GPT-4, as well as continue to use the standard GPT-3.5.

GPT-4 is more powerful but has a longer response time and a limit of 50 uses per 3 hours, which might not be ideal for frequent users.

Plus subscribers can also create images directly within the ChatGPT environment using DALL-E 3.

Perhaps the appeal of ChatGPT Plus is the ability to use custom GPTs. With Plus, you can access the ChatGPT Store and hundreds of GPTs created for specific tasks.

How to Use the ChatGPT GPT Store

The ChatGPT Store is a relatively new feature for ChatGPT Plus users, offering many practical GPTs to enhance your experience with ChatGPT.

If you find using prompts difficult or want to make the most of ChatGPT’s capabilities but don’t know how, visiting this store could be very helpful.

For example, Logo Creator allows you to generate logos directly from ChatGPT in seconds. GPT Consensus is one of the popular GPTs, capable of searching over 200 million academic papers and providing relatively accurate references.

Using GPTs from the ChatGPT Store is as simple as clicking on the app you like and starting to use it. You can also use the search function to narrow down what you’re looking for and check the star ratings—some GPTs are clearly better than others, and higher ratings can help you find GPTs truly worth your time.

Depending on your level of use with ChatGPT, you can even create and sell your own GPTs on the platform.
